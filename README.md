# HTOP Dashboard

HTOP Dashboard is a web-based application for monitoring CPU and RAM usage in real-time. It consists of a **Rust backend** and a **React TypeScript frontend**.

## Features

- **Real-time Monitoring**: Displays CPU and RAM usage with live updates via WebSocket.
- **Customizable Refresh Rate**: Adjust the frequency of updates.
- **Responsive Design**: Built with TailwindCSS for a modern and clean UI.

## Prerequisites

- **Docker** (Containerized deployment)

## Getting Started

Simply clone the repo and run `docker compose up --build -d`
